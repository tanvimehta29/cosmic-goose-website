Welcome to the team! This guide will help you get started and become a productive team member.

## Your First Week

### Day 1: Orientation
- Meet the team and mentors
- Tour the workspace
- Review safety guidelines
- Get added to communication channels

### Day 2-3: Training
- Complete safety training
- Learn basic tool usage
- Shadow experienced members
- Start on a beginner project

### Day 4-5: Integration
- Join a subteam
- Start contributing to tasks
- Ask lots of questions!

## Subteams

### Mechanical
Build and maintain the robot's physical structure.
- **Skills you'll learn:** CAD, fabrication, assembly
- **Tools:** Drills, saws, 3D printers

### Programming
Write software that controls the robot.
- **Skills you'll learn:** Java/Python, version control, debugging
- **Tools:** VS Code, GitHub

### Electrical
Handle wiring, sensors, and electronics.
- **Skills you'll learn:** Wiring, soldering, circuit design
- **Tools:** Multimeter, crimping tools

### Business
Handle sponsorships, marketing, and awards.
- **Skills you'll learn:** Communication, writing, design
- **Tools:** Presentation software, social media
